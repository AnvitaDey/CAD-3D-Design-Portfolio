# CAD & 3D Design Portfolio

This repository contains my CAD designs and 3D printed models developed using Fusion 360 for prototyping and real-world applications.

I enjoy designing components that are not just visually appealing, but also practical.

---

## 🚀 Overview

This portfolio includes a collection of:

* Mechanical components and assemblies
* Robot chassis and structural parts
* Functional prototypes for real-world use
* Design experiments and concept models

All designs are created with a focus on **structural efficiency, manufacturability, and usability**.

---

## 🧠 Design Approach

* Design for **real-world constraints** (strength, stability, space)
* Ensure **printability** and ease of assembly
* Focus on **modular and maintainable designs**
* Iterate based on testing and practical feedback

---

## 📂 Projects

Each folder contains:

* 📸 Renders / images of the design
* 🧩 STL files for 3D printing
* 📝 Short description of the design

---

## 🛠️ Tools & Workflow

* Fusion 360 (CAD Design)
* Cura (3D Printing – slicing, STL preparation)

Workflow:
CAD → STL → Slicing → 3D Printing → Iteration

Many of these designs are part of larger systems (such as robotics projects), while others are standalone prototypes and design explorations.
